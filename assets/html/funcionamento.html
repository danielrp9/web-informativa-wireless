Funcionamento

Por Raul Rodrigues

Para o funcionamento de uma rede wireless, são necessários alguns dispositivos que trabalham em conjunto para oferecer conexão rápida com número reduzido de cabos. Os dispositivos necessários cumprem funções como converter sinais, direcionar pacotes de dados, dentre outras funções.

Dispositivos como computadores, celulares, impressoras e outros, esses dispositivos são manipulados pelos usuários e enviam/recebem dados para o roteador.
